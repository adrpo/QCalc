QCalc.Units.Bases

Sets of base constants and units

Information

Some systems of units are not compatible with Units. Although the structure of the Units package is general, the models in Units require that the Faraday and gas constants are normalized to one. It follows that kB = q. This is not the case for the Planck, Rydberg, and Natural systems of units [http://en.wikipedia.org/wiki/Natural_units].

The quasi-SI sets in this package are named by listing (in alphabetical order) the two units (out of A, cd, K, kg, m, mol, and s) that are not normalized for the sake of normalizing the Faraday and gas constants. There are eight possible systems of this type (SIAK, SIAm, SIAs, SIKmol, SIKs, SImmol, SIms, and SImols).

For more information, please see the documentation for the Units package.

QCalc.Units.Bases.Gaussian

Base constants and units for Gaussian units (kA = kC = 1)

Information

Gaussian systems of units impose that:

  1. the magnetic force constant is one (kA = 1) (⇒ RK/c = 2π/α) and
  2. the electric force constant is one (kC = 1) (⇒ RK c = 2π/α).

This implies that c = 1 and RK = 2π/α.

The Gaussian conditions are not sufficient to fully establish the values of the base constants and units of the Units package. Gaussian units encompass other systems of units.

For more information, please see the documentation for the Units package.

Extends from Base (Base constants and units).

Modelica definition

record Gaussian "Base constants and units for Gaussian units (kA = kC = 1)
"
  extends Base(final c=1,final R_K=25812.8074434/299792458e-7);
end Gaussian;

QCalc.Units.Bases.Hartree

Base constants and units for Hartree atomic units

Information

Stoney units impose that:

  1. the elementary charge is one (q = 1),
  2. Planck's constant is two pi (h = 2π),
  3. the electric force constant is one (kC = 1), and
  4. the mass of an electron is one (4π  kA R∞ q2 = α3).

Please see the documentation for the Units package.

Extends from Base (Base constants and units).

Modelica definition

record Hartree "Base constants and units for Hartree atomic units" extends Base( final R_inf=299792458e-7*pi/25812.8074434, final c=1/(2*R_inf), final k_J=2, final R_K=1, k_Aprime=2*pi*R_K/c); end Hartree;

QCalc.Units.Bases.LH

Base constants and units for Lorentz-Heaviside units (μ0 = ε0 = 1)

Information

Lorentz-Heaviside systems of units impose that:

  1. the magnetic constant is one (μ0 = 1) (⇒ RK/c = 1/(2α)) and
  2. the electric constant is one (ε0 = 1) (⇒ RK c = 1/(2α)).

This implies that c = 1 and RK = 1/(2α).

The Lorentz-Heaviside conditions are not sufficient to fully establish the values of the base constants and units of the Units package. Lorentz-Heaviside units encompass other systems of units.

For more information, please see the documentation for the Units package.

Extends from Base (Base constants and units).

Modelica definition

record LH "Base constants and units for Lorentz-Heaviside units (μ0 = ε0 = 1)
"
  extends Base(final c=1,final R_K=25812.8074434/(4*pi*299792458e-7));
end LH;

QCalc.Units.Bases.Stoney

Base constants and units for Stoney units

Information

Stoney units are Gaussian units (kA = kC = 1) which also impose that:

  1. the elementary charge is one (q = 1) and
  2. the gravitational constant is one (although not included in the Units package).

For more information, please see the documentation for the Units package.

Extends from Gaussian (Base constants and units for Gaussian units (kA = kC = 1)).

Modelica definition

record Stoney "Base constants and units for Stoney units" extends Gaussian(final k_J=4*pi*299792458e-7/25812.8074434); end Stoney;

QCalc.Units.Bases.SIKmol

Base constants and units for SI with kF and R normalized instead of K and mol

Information

The values of the un-normalized SI base units are (see Resources/unit-systems.cdf):

  • K ≈ 8.61733e-5
  • mol ≈ 96485.3

For more information, please see the documentation for the Units package.
